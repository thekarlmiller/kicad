THE IoT Home Automation board

ESP32 based modules carrier board

This project template is a starting point for creating a device to automate a portion of your home. This device will be able to include one or more modules from THE IoT Home Automation project. The modules provide a sub-circuit of sorts that, each, provide a single function (such as a sensor or motor driver). The modules all are designed to work together homogenously in a platform based on a ESP32 MCU, 3.3V or 5V operating voltages. More information can be found at THEIoTHomeAutomation.
